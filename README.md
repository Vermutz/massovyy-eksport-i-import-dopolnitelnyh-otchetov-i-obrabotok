# Массовый экспорт и импорт дополнительных отчетов и обработок (БСП, 8.3)

## Описание

Внешняя обработка для массовой выгрузки и загрузки элементов справочника `ДополнительныеОтчетыИОбработки`. Позволяет быстро извлекать файлы `.epf` и `.erf` из базы, вносить в них изменения и загружать обратно, минуя ручное скачивание каждого файла по отдельности.

Инструмент поддерживает гибкую фильтрацию по виду, назначению и маске имени. Работает в любых конфигурациях 1С, построенных на базе Библиотеки стандартных подсистем (БСП).

## Возможности

**Отборы** (работают в комбинации по логике «И»):
- **По виду объекта:** отчёт, обработка, печатная форма, заполнение объекта.
- **По назначению:** конкретный документ, справочник, регистр.
- **По группе справочника:** с учётом вложенности.
- **По маске имени файла:** `*` — любая последовательность, `?` — один символ.
- **Флаг «Только опубликованные»:** выгружает только объекты с публикацией «Используется».
- 
- **Новое в версии 1.1.0:**
- **Сохранение иерархии групп** — при экспорте файлы сохраняются во вложенных папках, соответствующих структуре справочника (по чекбоксу «Сохранять иерархию»)
- **Раздельные каталоги** — отдельные поля для каталога экспорта и импорта, чтобы исходные файлы оставались нетронутыми
- **Рекурсивный поиск** — при импорте обработка автоматически находит файлы во всех подкаталогах

**Режимы работы:**
- **Заполнение таблицы:** замена списка или добавление к существующим (без создания дублей).
- **Импорт:** перезапись хранилища найденного элемента. Если элемент с таким `ИмяФайла` отсутствует в базе — файл пропускается с записью в лог ошибок.

## Установка

1. Скачайте файл обработки `МассовыйЭкспортИмпортДопОтчетовИОбработок.epf`.
2. Откройте его в 1С:Предприятие через меню **Файл → Открыть** или двойным кликом по файлу.
3. Обработка готова к работе.

## Использование

Работа с инструментом делится на два основных сценария.

### Сценарий 1: Массовая выгрузка

1. Откройте обработку.
2. В поле **«Отбор вид»** выберите нужный тип (например, «Печатная форма»).
3. При необходимости уточните отбор по **«Назначению»** (например, «Приём на работу (Документ)») или укажите **«Маску имени файла»**.
4. Нажмите кнопку **«Заполнить»** — в таблицу попадут все объекты, удовлетворяющие условиям.
5. Выберите **«Каталог выгрузки»** через диалог выбора папки.
6. Нажмите **«Выгрузить»**. Файлы `.epf`/`.erf` будут сохранены в указанную директорию.

### Сценарий 2: Обратная загрузка (Импорт)

1. Внесите правки в выгруженные ранее файлы (через Конфигуратор или ваш любимый IDE/редактор кода).
2. Поместите исправленные файлы в каталог, из которого будет выполняться загрузка.
3. В обработке укажите этот каталог.
4. Нажмите **«Загрузить»**. Хранилища элементов справочника обновятся автоматически.

## Требования

- Платформа 1С:Предприятие 8.3 (управляемые формы).
- Конфигурация на БСП со справочником `ДополнительныеОтчетыИОбработки` (БП 3.0, ЗУП 3.1, УТ 11, ERP, УНФ и другие).
- Права на изменение справочника дополнительных отчётов и обработок.

## Известные ограничения

- **Не выгружает группы**, только конечные элементы справочника.
- **Поиск файлов при импорте** осуществляется строго по реквизиту `ИмяФайла`. Если реквизит у элемента пуст — объект будет пропущен.
- **Требуется доступ** к файловой системе клиента для чтения/записи в каталоги выгрузки и загрузки.
- **Новые элементы не создаются** — при импорте обновляются только те элементы справочника, которые уже существуют в базе.
